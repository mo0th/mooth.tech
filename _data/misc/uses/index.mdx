## Software

### Editor & Teminal

My main editor is [Visual Studio Code](https://code.visualstudio.com/) with these extensions:

- [Dracula Official](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula)
  (theme)
- [Vim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim)

When I'm stuck in a terminal, or just hacking on some scripts I use [neovim](https://neovim.io/)
with [this](https://github.com/soorria/wsl-dotfiles/blob/master/config/nvim/init.vim) `init.vim`,

I use WSL through [Windows Terminal](https://github.com/Microsoft/Terminal) and use Bash as my
shell. For my shell prompt I use [Starship](https://starship.rs/) with
[this](https://github.com/soorria/wsl-dotfiles/blob/master/config/starship.toml) config.

For my terminal and editor, the font I use is [JetBrains Mono](https://www.jetbrains.com/lp/mono/).
All my current-ish dotfiles are [here](https://github.com/soorria/wsl-dotfiles), and the scripts I
use are [here](https://github.com/soorria/scripts).
