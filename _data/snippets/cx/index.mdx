A tiny helper function to conditionally join classnames. Simplified version of
JedWatson's [classnames](https://github.com/JedWatson/classnames) and lukeed's
[clsx](https://github.com/lukeed/clsx).

```typescript title="cx.ts" caption="yo"
export const cx = (...classes: (string | boolean | null | undefined)[]) =>
  classes.filter(cls => typeof cls === 'string').join(' ')
```
